# User Stories

---

## Authentication

- [ ] As a new member I want to be able to create an account to secure my data
- [ ] As a member I want to be able to log in to access my account
- [ ] As a member I want to reset my password to recover access to my account
- [ ] As a member I want to change my password to keep my account secure
- [ ] As a member I want to edit my profile information to keep my account up to date
- [ ] As a member I want to be able to connect through SSO (Future)
- [ ] As a member I want to set a 2FA to secure my account (Future)

---

## Boards

- [ ] As a superadmin I want to create a board to organise my projects
- [ ] As a superadmin I want to be able to create multiple boards to organise my projects/ideas
- [ ] As a superadmin or admin I want to be able to assign members to a board
- [ ] As a superadmin or admin I want to be able to remove members from a board to secure my application
- [ ] As a superadmin I want to be able to create custom roles to extend the default permission system
- [ ] As a superadmin I want to be able to assign permissions to a custom role to control precisely what its members can do
- [ ] As a superadmin or admin I want to be able to assign roles to members to control what they can do
- [ ] As a superadmin or admin I want to edit a board name and description to keep it up to date
- [ ] As a superadmin I want to delete a board to remove projects that are no longer active
- [ ] As a superadmin or admin I want to archive a board to keep it accessible without cluttering the workspace
- [ ] As a superadmin or admin I want to generate a shareable invite link to a board to allow members to join without going through the superadmin
- [ ] As a superadmin or admin I want the invite link to expire after a set time or number of uses to keep the board secure
- [ ] As a member I want to join a board via a shareable link to access it without the superadmin manually assigning me
- [ ] As a member I want to see the boards I have joined to access my work quickly

---

## Columns

- [ ] As a superadmin or admin I want to create custom columns to adapt the board to my workflow
- [ ] As a superadmin or admin I want to edit a column name to rename it without deleting it
- [ ] As a superadmin or admin I want to reorder columns to adapt the board layout to my workflow
- [ ] As a superadmin or admin I want to delete a column to remove steps that are no longer needed

---

## Cards

- [ ] As a member I want to create cards to display my idea(s) if my role allows it
- [ ] As a member I want to move a card from one column to another to update its progress if my role allows it
- [ ] As a member I want to add a description to a card to give more context
- [ ] As a member I want to add a due date to a card to track deadlines
- [ ] As a member I want to edit or remove a due date on a card to update the timeline
- [ ] As a member I want to assign a card to a member to know who is responsible
- [ ] As a member I want to add a tag to a card to categorise it
- [ ] As a member I want to remove a tag from a card to keep it organised
- [ ] As a member I want to archive a card to keep the board clean without losing data if my role allows it
- [ ] As a member I want to edit a card title to correct or update it
- [ ] As a member I want to delete a card to remove it permanently if it is no longer needed
- [ ] As a member I want to reorder cards within a column to prioritise my work
- [ ] As a member I want to be able to leave a comment under a card to share my thoughts (Future)
- [ ] As a member I want to be able to add a file to a card (Future)
- [ ] As a member I want to be able to see the history of a card to know what has been changed (Future)

---

## Tags

- [ ] As a superadmin or admin I want to create tags to categorise cards on my board
- [ ] As a superadmin or admin I want to edit a tag to rename or recolor it
- [ ] As a superadmin or admin I want to delete a tag to keep the tag list clean

---

## Notifications

- [ ] As a superadmin or admin I want to be able to set webhooks to send notifications when a change happens to a board
- [ ] As a member I want to receive a notification when a card is assigned to me to stay informed of my responsibilities

---

## Administration

- [ ] As a superadmin I want to manage all members of the instance to maintain control over who has access
- [ ] As a superadmin I want to edit a member account to correct information or reset access
- [ ] As a superadmin I want to delete a member account to remove someone from the instance permanently
- [ ] As a superadmin I want to disable public registration to prevent unknown members from creating accounts
- [ ] As a superadmin I want to configure the instance domain name to enable shareable invite links with a valid URL
- [ ] As a superadmin I want to configure an SMTP server to enable email notifications from my instance
